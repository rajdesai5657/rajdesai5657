<summary><b>🐍 Contribution Snake (setup instructions)</b></summary>
<br/>

To render an animated snake eating your contribution graph, add a GitHub Actions workflow (`.github/workflows/snake.yml`) using `Platane/snk`, then embed the generated SVG below:

```yaml
name: generate snake
on:
  schedule:
    - cron: "0 0 * * *"
  workflow_dispatch: {}
  push:
    branches: [ main ]

jobs:
  generate:
    runs-on: ubuntu-latest
    steps:
      - uses: Platane/snk@v3
        with:
          github_user_name: rajdesai5657
          outputs: dist/github-snake.svg
      - uses: actions/upload-artifact@v4
        with:
          name: github-snake
          path: dist/github-snake.svg
      - uses: crazy-max/ghaction-github-pages@v4
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```
